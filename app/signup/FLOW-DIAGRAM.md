## Key Interactions

### Step 1: Choose Path
- **Action**: Click card
- **Effect**: Set path, advance to Step 2 (or redirect for returning members)
- **Animation**: Scale up on hover, slide in transition

### Step 2: Member Info
- **Action**: Enter date of birth
- **Effect**: Auto-detect minor status, show/hide parent fields
- **Animation**: Parent section slides in smoothly

### Step 3: Choose Plan
- **Action**: Click pricing card
- **Effect**: Highlight selected, show family section if Family Plan
- **Animation**: Card scale up, checkmark appears

### Step 4: Waiver
- **Action**: Draw signature
- **Effect**: Canvas captures strokes, exports base64 PNG
- **Validation**: Must agree + sign + type name before submitting

## State Persistence

```javascript
// On every form change:
useEffect(() => {
  if (formData.path) {
    localStorage.setItem('fortSignupProgress', JSON.stringify(formData));
  }
}, [formData]);

// On mount:
useEffect(() => {
  const saved = localStorage.getItem('fortSignupProgress');
  if (saved) {
    const parsed = JSON.parse(saved);
    setFormData(parsed);
    // Jump to correct step
    if (parsed.signatureData) setCurrentStep(4);
    else if (parsed.membershipPlan) setCurrentStep(3);
    else if (parsed.firstName) setCurrentStep(2);
  }
}, []);
```
